# Simple Linear Regression – Marketing ROI Analysis

## Project Overview

This project analyzes a marketing dataset to determine which advertising channel (TV, Radio, or Social Media) has the strongest impact on Sales. Using **Simple Linear Regression** with Python's `statsmodels` library, I built a predictive model, validated regression assumptions, and formulated a data-driven recommendation for marketing budget allocation.

### Key Objectives:
- Load and clean the marketing dataset
- Identify the variable most correlated with Sales
- Build and validate an OLS regression model
- Interpret R-squared, coefficients, and p-values
- Provide ROI-based budget recommendation

### Dataset:
The data contains spend on three marketing channels (TV, Radio, Social Media) and resulting Sales figures across multiple campaigns.
